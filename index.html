  • This map project was created using the mapbox API and integrated with the flickr API. Using the photo.search and photo.get.geo methods we are able to retrieve public photos within a bounding area defined by the initial mapbox viewport with a degree of accuracy representing how wide want to search for public photos. The data retrieved from flickr was retrieved using jQuery ajax calls and the data was parsed to JSON allowing us to rebuild the flickr info into the mapbox markers. If a new image is added into the flickr database in our viewport we can see it just by clicking show flickr photos again. Rather than using backbone to handle our delete and addFaves method calls to store/retrieve data into mongoDB we simply used two separate get requests.
